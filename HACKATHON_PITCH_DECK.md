# Hackathon Pitch Deck (10 Slides)

## Slide 1 — Title
**Cyber Chaukidaar**
- Full‑stack cybersecurity suite
- Browser extension + Web dashboard + USB Vault
- Team / Hackathon / Date

## Slide 2 — Problem
- Users face phishing, trackers, ads, weak passwords
- Data breaches and credential reuse are rising
- Security tools are fragmented and hard to use

## Slide 3 — Solution Overview
- Real‑time protection across web
- Unified system: Extension ↔ Web ↔ USB Vault
- Simple UX with strong security defaults

## Slide 4 — Browser Extension Features
- Site safety scoring + overlays
- Ad & tracker blocking (DNR rules)
- Password manager with strength checks
- HTTP blocking with advanced override

## Slide 5 — Web Dashboard Features
- Live extension stats (ads/trackers/sites/passwords)
- Cyber hygiene score (computed from stats)
- Quick actions: Vault, Breach Check, AI Coach
- Threat log & system stats

## Slide 6 — USB Vault (Offline Security)
- AES‑256‑GCM encryption
- Hardware‑backed, portable
- Two‑way sync with extension
- Zero‑trust offline storage

## Slide 7 — Sync & Architecture
- Extension → CustomEvent → Web dashboard
- Vault ↔ Extension sync flow
- Secure local storage + encryption boundaries

## Slide 8 — Demo Flow (Live)
1. Visit site → safety overlay
2. Ad/Tracker counters increase
3. Save password + strength rating
4. Sync to USB Vault
5. Dashboard updates in real time

## Slide 9 — Impact
- Faster threat detection
- Reduced exposure to ads/trackers
- Safer password handling
- Offline security for critical data

## Slide 10 — Roadmap & Ask
- Add enterprise policy controls
- Broader filter packs / ML‑based phishing
- Mobile companion + backup
- Ask: Support & adoption
